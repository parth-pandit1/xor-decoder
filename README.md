# XOR Decoder 

Python tool for detecting and decoding XOR-encrypted strings 
commonly found in malware samples during static analysis.

## What it does
- Decodes XOR-encrypted byte sequences
- Identifies encryption keys
- Used in malware static analysis workflow

## Usage
```bash
python disasm.py
```

## Skills Demonstrated
- Assembly language basics
- Malware string obfuscation techniques
- Static analysis methodology
